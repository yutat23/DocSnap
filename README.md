# DocSnap

紙のドキュメントをスマホで撮影して、きれいに整形するWebアプリ。

## 使い方

1. 画像を選択
2. 四隅のマーカーをドラッグしてドキュメント範囲を指定
3. 整形された画像を確認・ダウンロード

## 開発

```bash
npm install
npm run dev
```

[http://localhost:3000](http://localhost:3000) で開く。
